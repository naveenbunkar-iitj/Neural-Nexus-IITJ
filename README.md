# AI Decision Engine (PS-9)

## Problem Statement
Automation of Complex Decision-Making using AI

## Architecture
Frontend (Streamlit) → Backend (FastAPI) → ML Model

## Features
- AI-based decision system
- Risk scoring
- Explainable outputs
- API-based architecture

## Run Locally

### Backend:
uvicorn backend.api:app --reload

### Frontend:
streamlit run frontend/app.py

## Deployment
- API: Render
- Frontend: Streamlit Cloud

## Tech Stack
Python, FastAPI, Streamlit, Scikit-learn
